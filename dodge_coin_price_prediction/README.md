# Dogecoin Price Prediction with Machine Learning

![Dogecoin Price Prediction](https://images.unsplash.com/photo-1544476915-7e7b384f6bfd)

Dogecoin Price Prediction with Machine Learning is a Python application that utilizes machine learning techniques to forecast the price pattern of Dogecoin cryptocurrency. This project aims to provide traders and investors with valuable insights into potential price movements, enabling informed decision-making in the volatile cryptocurrency market.

## Overview

Dogecoin is a unique cryptocurrency that originated as a meme and gained popularity among crypto enthusiasts. Despite its humorous origins, Dogecoin has become a significant player in the cryptocurrency market, attracting attention from traders and investors.

The Dogecoin Price Prediction feature leverages historical price data and machine learning algorithms to forecast future price trends. By analyzing factors such as volume, price volatility, and market sentiment, the application generates predictions that can assist traders in making informed decisions.

## Installation

1. Clone the repository:

git clone https://github.com/AliRezaBmeDu/machine_learning_projects.git

2. Navigate to the directory
cd machine_learning_projects/dodgecoin

3. Install the necessary packages
pip install -r requirements.txt


## Usage

1. Download the Dogecoin price dataset (`DOGE-USD.csv`) from [here](https://drive.google.com/file/d/1d7pSL3uM0W-82aW63V37OVBacHUZ__hr/view?usp=share_link) and place it in the project directory.

2. Run the python file `price_prediction.oy` to perform data preprocessing, model training, and prediction generation.

3. Follow the instructions in the notebook to analyze the results and interpret the predictions.

## Documentation

The project documentation includes the following components:

- **Data Preprocessing:** Explains the steps involved in preparing the dataset for analysis, including handling missing values and converting date formats.

- **Exploratory Data Analysis (EDA):** Provides insights into the dataset through visualization and correlation analysis, highlighting key factors influencing Dogecoin price.

- **Model Development:** Describes the implementation of the SARIMAX (Seasonal Autoregressive Integrated Moving Average with exogenous variables) model for time series analysis.

- **Evaluation:** Evaluates the performance of the trained model and interprets the prediction results.
